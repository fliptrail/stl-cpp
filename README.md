# stl-cpp
Reimplementing the C++ Standard Template Library (STL) 

### To compile:
Works on `g++ (Ubuntu 9.3.0-17ubuntu1~20.04) 9.3.0`
```
g++ test.cpp -o test
```

### To run:
```
./test
```
